# Awwwwards Design Guide

Projekt opiera się na zasadach:

- immersive UI (duże media, mało tekstu)
- motion design (animacje)
- dark luxury style
- gesture navigation

1. FILOZOFIA DESIGNu

Nowoczesne aplikacje nie wyglądają jak klasyczne portale.

Najlepsze aplikacje:

- Instagram
- TikTok
- Airbnb

stosują zasadę:

Content First UI

czyli:

- 90% ekran = content
- 10% = interfejs

UI ma nie przeszkadzać.

1. GŁÓWNY STYL WIZUALNY

Tryb

🌙 Dark Mode First

Powód:

- media wyglądają lepiej
- bardziej premium
- mniejsze zmęczenie oczu

Paleta kolorów

- Background: #0B0B0F
- Card: #15151A
- Primary: #FF2E63
- Secondary: #08D9D6
- Accent: #FFD166
- Text: #FFFFFF

Efekt:

dark + neon + luxury

Gradient marki

Najważniejszy element wizualny:

`#FF2E63` -> `#FF7A18` -> `#FFD166`

Używany w:

- przyciskach
- rankingach
- elementach premium

1. TYPOGRAFIA

Font:

Inter

Rozmiary:

- H1 32px
- H2 24px
- Body 16px
- UI labels 14px

Dlaczego Inter?

- nowoczesny
- czytelny
- używany przez wiele startupów
