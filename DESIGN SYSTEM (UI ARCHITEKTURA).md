# DESIGN SYSTEM (UI ARCHITEKTURA)

To fundament aplikacji. Największe firmy (Airbnb, Apple, Stripe) budują spójny system UI, dzięki czemu aplikacja wygląda profesjonalnie.

## Styl wizualny

Nowoczesny styl:

- dark mode jako domyślny
- duże zdjęcia
- minimalizm
- duże przyciski

Inspiracje UX:

- TikTok
- Instagram
- Twitch
- OnlyFans

## Paleta kolorów

Podstawowe kolory aplikacji:

- Primary: #FF2E63
- Secondary: #08D9D6
- Background: #0F0F10
- Card: #1A1A1D
- Text: #FFFFFF

Akcent:

- Gold / Premium: #FFD166

## Typografia

Nowoczesna i czytelna.

- Heading: Inter Bold
- Body: Inter Regular
- UI labels: Inter Medium
